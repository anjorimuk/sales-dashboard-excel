# sales-dashboard-excel
Sales performance analysis and interactive dashboard using Excel.
**Sales Performance Analysis & Dashboard (Excel)**
This project analyzes sales data to identify key revenue drivers across categories, brands, stores, and time.

**Tools Used:**
Excel (Power Query, PivotTables, PivotCharts, VLOOKUP)

**Key Insights:**
1. Mountain Bikes generate highest revenue
2. Trek is the leading brand
3 .Baldwin Bikes is the top-performing store
4. Sales peak in April, indicating seasonality
